ArXiv Writer Documentation
==========================

ArXiv Writer is a standalone Python package for AI-assisted academic paper generation in arXiv format. 
It provides a clean, configurable API for generating academic papers using Large Language Models (LLMs) 
with support for multiple providers, template-based content generation, and publication-ready LaTeX output.
